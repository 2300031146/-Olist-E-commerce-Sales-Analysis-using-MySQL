# 📖 Project Overview

This project demonstrates a complete SQL-based analysis of the **Olist Brazilian E-Commerce Dataset**. The primary objective was to analyze customer behavior, sales performance, payment trends, delivery performance, and revenue distribution using SQL.

The project follows a real-world Data Analytics workflow:

**Raw Dataset → Database Exploration → Data Quality Validation → Business Analysis → Business Insights**

This project helped strengthen practical SQL skills while solving real business problems commonly encountered by Data Analysts.

---

# 🎯 Project Objectives

- Understand the structure of an E-commerce database
- Perform data quality validation
- Calculate key business metrics (KPIs)
- Analyze customer purchasing behavior
- Identify high-performing sellers and products
- Analyze payment trends
- Evaluate delivery performance
- Generate business insights using SQL
